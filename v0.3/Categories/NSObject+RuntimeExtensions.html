addClassMethodIfNeedWithSelector:toClass:

Adds a static method to the targetClass if it does not have one.

+ (BOOL)addClassMethodIfNeedWithSelector:(SEL)selector toClass:(Class)targetClass

Parameters

selector

A selector of the current class to be executed with the targetClass

targetClass

A class that gets a new static method.

Return Value

YES - In case a new method has been added to the targetClass NO - In case a method already existed. NO - In case of any objc-runtime errors.

Discussion

[ ClassA addClassMethodIfNeedWithSelector: @selector( doWork )
                                  toClass: [ ClassB class ] ];

[ ClassB doWork ]; // Will call +[ ClassA doWork ]

Adds a static method to the targetClass if it does not have one.

+ (BOOL)addClassMethodIfNeedWithSelector:(SEL)selector toClass:(Class)targetClass newMethodSelector:(SEL)newSelector

Parameters

selector

A selector of the current class to be executed with the targetClass

targetClass

A class that gets a new static method

newSelector

A method on the targetClass to be invoked by the end user.

Return Value

YES - In case a new method has been added to the targetClass NO - In case a newSelector method already existed. NO - In case of any objc-runtime errors.

Discussion

[ ClassA addClassMethodIfNeedWithSelector: @selector( doWork )
                                  toClass: [ ClassB class ]
                        newMethodSelector: @selector( doWorkFooBar ) ];

[ ClassB doWork       ]; // Throws "Does not recognize selector" exception

[ ClassB doWorkFooBar ]; // Will call +[ ClassA doWork ]

hookClassMethodForClass:withSelector:prototypeMethodSelector:hookMethodSelector:

Creates a hook for the existing static method of a given class. Depending on your impementation, the original method may either be executed or not. By default the original method is replaced.

+ (void)hookClassMethodForClass:(Class)targetClass withSelector:(SEL)targetSelector prototypeMethodSelector:(SEL)prototypeSelector hookMethodSelector:(SEL)hookSelector

Parameters

targetClass
  • a class for which the hook is being set up. Both targetSelector andhookSelector must be implemented for it.
targetSelector
  • a static method of the targetClass to be replaced
prototypeSelector
  • a static method of “self” to be executed instead of “targetSelector”
hookSelector
  • a static method of “self” where old implementation will be placed. It may be invoked from “prototypeSelector”
