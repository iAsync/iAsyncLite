asyncOperation:withOptionalStartActionBlock:optionalFinishActionBlock:

Subscribes an action block to operation start and a callback block to operation completion. Same as [JDoneCallbackAsyncOperationHooks asyncOperation:withStartActionBlock:finishCallbackBlock:] but hook blocks may be nil

+ (JFFAsyncOperation)asyncOperation:(JFFAsyncOperation)loader withOptionalStartActionBlock:(JFFSimpleBlock)startBlock optionalFinishActionBlock:(JFFSimpleBlock)doneBlock

Parameters

startBlock

An action block to subscribe for operation start.

doneBlock

A block callback to subscribe for completion. The original callback will be executed.
