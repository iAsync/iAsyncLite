Instance Methods

balancedLoaderWithLoader:

Wraps the loader and associates it with the current queue.

- (JFFAsyncOperation)balancedLoaderWithLoader:(JFFAsyncOperation)loader

Parameters

loader

An asynchronous operation to execute in a queue.

Return Value

A wrapped asynchronous operation.

The queue workflow : 1. Once invoked, the resulting operation is added to the pending loaders list of the corresponding queue. 2. The operation stays pending while the limitCount is exceeded. 3. When active loaders count is less than limitCount the loader operation is moved to the active loaders list. 4. The operation is executed by the dispatch_async() invocation. 5. The operation is finished and the user gets proper callback notifications.
