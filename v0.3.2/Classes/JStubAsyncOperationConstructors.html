asyncOperationWithCancelFlag:

Creates an asynchronouys operation that gets cancelled with the specified flag. The operation will be cancelled instead of being executed.

+ (JFFAsyncOperation)asyncOperationWithCancelFlag:(BOOL)canceled

Parameters

canceled

YES to cancel both the result operation and related ones. NO to unsubscribe the operation.

Return Value

An asynchronous operation.

Note : such operations should be used in unit tests only.
